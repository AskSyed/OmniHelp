# Omni-Help Frontend

Angular 18 frontend for the Omni-Help intelligent customer support platform.

## Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm start
   ```

3. Open browser:
   Navigate to http://localhost:4200

## Project Structure

- `src/app/components/`: UI components (Chat, Document Upload, Order Management)
- `src/app/services/`: API service layer
- `src/environments/`: Environment configuration

## Features

- **Chat Interface**: Interactive chat with intent-based routing visualization
- **Document Upload**: Drag-and-drop PDF upload with processing status
- **Order Management**: Create and search orders with customer information
